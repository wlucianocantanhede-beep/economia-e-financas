Como ajustar a variância de uma carteira para um valor mínimo

Antes de ler este artigo, é ideal que veja o artigo “Como zerar a variância de uma carteira composta por dois ativos”, disponível no link a seguir: https://wlucianocantanhede-beep.github.io/economia-e-financas/como-zerar-a-variancia-de-uma-carteira-composta-por-dois-ativos.html

A expressão de cálculo do desvio-padrão de uma carteira composta por nn ativos é a seguinte:

σp=(∑i=1n∑j=1nwiwjCorri,jσiσj)1/2\sigma_p = \left( \sum_{i=1}^{n} \sum_{j=1}^{n} w_i w_j \,\mathrm{Corr}_{i,j}\, \sigma_i \sigma_j \right)^{1/2}

Suponha uma carteira composta por dois ativos, XX e YY.

Sabe-se que Corr(X,Y)=Cov(X,Y)desvio padrão(X)×desvio padrão(Y)\mathrm{Corr}(X,Y) = \frac{\mathrm{Cov}(X,Y)}{\text{desvio padrão}(X) \times \text{desvio padrão}(Y)}

O retorno da carteira é:

C=w1X+w2YC = w_1 X + w_2 Y

O retorno esperado da carteira é:

E(C)=w1E(X)+w2E(Y)E(C) = w_1 E(X) + w_2 E(Y)

Onde w1w_1 e w2w_2 são pesos.

Para valores dados dos desvios padrão em cada ponto, suponha Corr(X,Y)=0\mathrm{Corr}(X,Y) = 0.

Veja que

Cov(C,C)=Var(C)\mathrm{Cov}(C,C) = \mathrm{Var}(C)

e

C=w1X+w2YC = w_1 X + w_2 Y

Var(C)=Var(w1X+w2Y)=w12Var(X)+w22Var(Y)+2w1w2Cov(X,Y)\mathrm{Var}(C) = \mathrm{Var}(w_1 X + w_2 Y) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y) + 2 w_1 w_2 \mathrm{Cov}(X,Y)

Como

Corr(X,Y)=Cov(X,Y)desvio padrão(X)×desvio padrão(Y)\mathrm{Corr}(X,Y) = \frac{\mathrm{Cov}(X,Y)}{\text{desvio padrão}(X) \times \text{desvio padrão}(Y)}

tem-se

Var(C)=w12Var(X)+w22Var(Y)+2w1w2Corr(X,Y)desvio padrão(X)desvio padrão(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y) + 2 w_1 w_2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)

Logo, quando Corr(X,Y)=0\mathrm{Corr}(X,Y)=0, o que sobra como risco da carteira é a expressão

w12Var(X)+w22Var(Y)w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y)

Minimizando o risco da carteira:

Var(C)=w12Var(X)+w22Var(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y)

Sob

w1+w2=1w_1 + w_2 = 1

tem-se

Var(C)=w12Var(X)+(1−w1)2Var(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + (1 - w_1)^2 \mathrm{Var}(Y)

Var(C)=w12Var(X)+(1−2w1+w12)Var(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + (1 - 2 w_1 + w_1^2)\mathrm{Var}(Y)

Var(C)=w12Var(X)+Var(Y)−2w1Var(Y)+w12Var(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + \mathrm{Var}(Y) - 2 w_1 \mathrm{Var}(Y) + w_1^2 \mathrm{Var}(Y)

Derivando:

dVar(C)dw1=2w1Var(X)−2Var(Y)+2w1Var(Y)=0\frac{d\,\mathrm{Var}(C)}{d w_1} = 2 w_1 \mathrm{Var}(X) - 2 \mathrm{Var}(Y) + 2 w_1 \mathrm{Var}(Y) = 0

2w1Var(X)=2Var(Y)−2w1Var(Y)2 w_1 \mathrm{Var}(X) = 2 \mathrm{Var}(Y) - 2 w_1 \mathrm{Var}(Y)

w1Var(X)=Var(Y)−w1Var(Y)w_1 \mathrm{Var}(X) = \mathrm{Var}(Y) - w_1 \mathrm{Var}(Y)

w1(Var(X)+Var(Y))=Var(Y)w_1 (\mathrm{Var}(X) + \mathrm{Var}(Y)) = \mathrm{Var}(Y)

w1=Var(Y)Var(X)+Var(Y)w_1 = \frac{\mathrm{Var}(Y)}{\mathrm{Var}(X) + \mathrm{Var}(Y)}

e

w2=Var(X)Var(X)+Var(Y)w_2 = \frac{\mathrm{Var}(X)}{\mathrm{Var}(X) + \mathrm{Var}(Y)}

Veja que, de fato, há minimização do risco, comprovada via segunda derivada:

d2Var(C)dw12=2Var(X)+2Var(Y)=2(Var(X)+Var(Y))\frac{d^2\,\mathrm{Var}(C)}{d w_1^2} = 2 \mathrm{Var}(X) + 2 \mathrm{Var}(Y) = 2 (\mathrm{Var}(X) + \mathrm{Var}(Y))

Se Var(X),Var(Y)>0\mathrm{Var}(X), \mathrm{Var}(Y) > 0, há minimização do risco, pois d2Var(C)dw12>0\frac{d^2\,\mathrm{Var}(C)}{d w_1^2} > 0

Se fosse máximo, a segunda derivada teria que ser menor do que 0.

O exposto acima é uma situação específica de minimização de risco em que Corr(X,Y)=0\mathrm{Corr}(X,Y)=0.

Pode-se estender o leque de possibilidades de minimização do risco da carteira a partir das expressões de cálculo dos pesos a seguir, válidas em situações em que

−1<Corr(X,Y)<1-1 < \mathrm{Corr}(X,Y) < 1

A variância da carteira é dada por:

Var(C)=w12Var(X)+w22Var(Y)+2w1w2Corr(X,Y)desvio padrão(X)desvio padrão(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y) + 2 w_1 w_2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)

Substituindo w2=1−w1w_2 = 1 - w_1:

Var(C)=w12Var(X)+(1−w1)2Var(Y)+2w1(1−w1)Corr(X,Y)desvio padrão(X)desvio padrão(Y)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + (1 - w_1)^2 \mathrm{Var}(Y) + 2 w_1 (1 - w_1)\mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)

Derivando e igualando a zero:

dVar(C)dw1=2w1Var(X)−2Var(Y)+2w1Var(Y)+2Corr(X,Y)desvio padrão(X)desvio padrão(Y)−4w1Corr(X,Y)desvio padrão(X)desvio padrão(Y)=0\frac{d\,\mathrm{Var}(C)}{d w_1} = 2 w_1 \mathrm{Var}(X) - 2 \mathrm{Var}(Y) + 2 w_1 \mathrm{Var}(Y) + 2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y) - 4 w_1 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y) = 0

Resolvendo:

w1=Var(Y)−Corr(X,Y)desvio padrão(X)desvio padrão(Y)Var(X)+Var(Y)−2Corr(X,Y)desvio padrão(X)desvio padrão(Y)w_1 = \frac{\mathrm{Var}(Y) - \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)} {\mathrm{Var}(X) + \mathrm{Var}(Y) - 2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)}

e

w2=Var(X)−Corr(X,Y)desvio padrão(X)desvio padrão(Y)Var(X)+Var(Y)−2Corr(X,Y)desvio padrão(X)desvio padrão(Y)w_2 = \frac{\mathrm{Var}(X) - \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)} {\mathrm{Var}(X) + \mathrm{Var}(Y) - 2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)}

Em síntese, sob a restrição w1+w2=1w_1 + w_2 = 1, é possível minimizar a variância de uma carteira composta por dois ativos quando

−1<Corr(X,Y)<1-1 < \mathrm{Corr}(X,Y) < 1

As expressões gerais dos cálculos dos pesos são:

w1=Var(Y)−Corr(X,Y)desvio padrão(X)desvio padrão(Y)Var(X)+Var(Y)−2Corr(X,Y)desvio padrão(X)desvio padrão(Y)w_1 = \frac{\mathrm{Var}(Y) - \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)} {\mathrm{Var}(X) + \mathrm{Var}(Y) - 2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)}

e

w2=Var(X)−Corr(X,Y)desvio padrão(X)desvio padrão(Y)Var(X)+Var(Y)−2Corr(X,Y)desvio padrão(X)desvio padrão(Y)w_2 = \frac{\mathrm{Var}(X) - \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)} {\mathrm{Var}(X) + \mathrm{Var}(Y) - 2 \mathrm{Corr}(X,Y)\, \text{desvio padrão}(X)\, \text{desvio padrão}(Y)}

Quando Corr(X,Y)=0\mathrm{Corr}(X,Y)=0, as expressões se reduzem a:

w1=Var(Y)Var(X)+Var(Y)w_1 = \frac{\mathrm{Var}(Y)}{\mathrm{Var}(X) + \mathrm{Var}(Y)}

e

w2=Var(X)Var(X)+Var(Y)w_2 = \frac{\mathrm{Var}(X)}{\mathrm{Var}(X) + \mathrm{Var}(Y)}
