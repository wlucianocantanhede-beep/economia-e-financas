Como zerar a variância de uma carteira composta por dois ativos

Suponha uma carteira composta por dois ativos, XX e YY. Nesse caso, a diversificação perfeita, em termos de correlação, requer uma correlação entre os ativos de −1-1.

Sabe-se que Corr(X,Y)=Cov(X,Y)σXσY\mathrm{Corr}(X,Y) = \frac{\mathrm{Cov}(X,Y)}{\sigma_X \, \sigma_Y}

O retorno da carteira é: C=w1X+w2YC = w_1 X + w_2 Y

O retorno esperado da carteira é: E(C)=w1E(X)+w2E(Y)E(C) = w_1 E(X) + w_2 E(Y)

Onde w1w_1 e w2w_2 são pesos.

Para valores dados dos desvios padrão em cada ponto, suponha Corr(X,Y)=−1=Cov(X,Y)σXσY\mathrm{Corr}(X,Y) = -1 = \frac{\mathrm{Cov}(X,Y)}{\sigma_X \, \sigma_Y}

Veja que Cov(C,C)=Var(C)\mathrm{Cov}(C,C) = \mathrm{Var}(C) e C=w1X+w2YC = w_1 X + w_2 Y

Var(C)=Var(w1X+w2Y)=w12Var(X)+w22Var(Y)+2w1w2Cov(X,Y)\mathrm{Var}(C) = \mathrm{Var}(w_1 X + w_2 Y) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y) + 2 w_1 w_2 \mathrm{Cov}(X,Y)

Como Corr(X,Y)=Cov(X,Y)σXσY\mathrm{Corr}(X,Y) = \frac{\mathrm{Cov}(X,Y)}{\sigma_X \, \sigma_Y}

Var(C)=w12Var(X)+w22Var(Y)+2w1w2Corr(X,Y)σXσY\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y) + 2 w_1 w_2 \mathrm{Corr}(X,Y)\,\sigma_X \sigma_Y

Logo, quando Corr(X,Y)=−1\mathrm{Corr}(X,Y) = -1, o valor de Var(C)\mathrm{Var}(C) diminui a uma magnitude de 2w1w2σXσY2 w_1 w_2 \sigma_X \sigma_Y mas o risco, Var(C)\mathrm{Var}(C), ainda existe se os pesos não respeitarem a proporção correta.

Veja que, se os pesos forem positivos, Corr(w1X,w2Y)=Corr(X,Y)\mathrm{Corr}(w_1 X, w_2 Y) = \mathrm{Corr}(X,Y) Nesse caso, a correlação é igual a Corr(X,Y)\mathrm{Corr}(X,Y); ou seja, os pesos não interferem na correlação.

Var(C)=w12Var(X)+w22Var(Y)+2w1w2Corr(X,Y)σXσY\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(Y) + 2 w_1 w_2 \mathrm{Corr}(X,Y)\,\sigma_X \sigma_Y

Quando Corr(X,Y)=−1\mathrm{Corr}(X,Y) = -1 e Var(X)=Var(Y)\mathrm{Var}(X) = \mathrm{Var}(Y),

Var(C)=w12Var(X)+w22Var(X)−2w1w2Var(X)\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + w_2^2 \mathrm{Var}(X) - 2 w_1 w_2 \mathrm{Var}(X)

Var(C)=Var(X)(w12+w22−2w1w2)\mathrm{Var}(C) = \mathrm{Var}(X)\,(w_1^2 + w_2^2 - 2 w_1 w_2)

Var(C)=Var(X)(w1−w2)2\mathrm{Var}(C) = \mathrm{Var}(X)\,(w_1 - w_2)^2

Isto é, sob essas condições, pesos iguais tornam a variância da carteira igual a zero.

Para variâncias iguais e Corr(X,Y)=1\mathrm{Corr}(X,Y) = 1,

Var(C)=Var(X)(w1+w2)2=Var(X)=Var(Y)\mathrm{Var}(C) = \mathrm{Var}(X)\,(w_1 + w_2)^2 = \mathrm{Var}(X) = \mathrm{Var}(Y)

Para variâncias iguais,

Var(C)=Var(X)(w12+w22+2w1w2Corr(X,Y))\mathrm{Var}(C) = \mathrm{Var}(X)\,(w_1^2 + w_2^2 + 2 w_1 w_2 \mathrm{Corr}(X,Y))

Var(C)Var(X)=w12+w22+2w1w2Corr(X,Y)\frac{\mathrm{Var}(C)}{\mathrm{Var}(X)} = w_1^2 + w_2^2 + 2 w_1 w_2 \mathrm{Corr}(X,Y)

Igualando Var(C)\mathrm{Var}(C) a zero,

0=w12+w22+2w1w2Corr(X,Y)0 = w_1^2 + w_2^2 + 2 w_1 w_2 \mathrm{Corr}(X,Y)

w12+w22=−2w1w2Corr(X,Y)w_1^2 + w_2^2 = -2 w_1 w_2 \mathrm{Corr}(X,Y)

(w1+w2)2−2w1w2=−2w1w2Corr(X,Y)(w_1 + w_2)^2 - 2 w_1 w_2 = -2 w_1 w_2 \mathrm{Corr}(X,Y)

(w1+w2)2=2w1w2(1−Corr(X,Y))(w_1 + w_2)^2 = 2 w_1 w_2 (1 - \mathrm{Corr}(X,Y))

Como w1+w2=1w_1 + w_2 = 1,

1=2w1w2(1−Corr(X,Y))1 = 2 w_1 w_2 (1 - \mathrm{Corr}(X,Y))

w1w2=12(1−Corr(X,Y))w_1 w_2 = \frac{1}{2(1 - \mathrm{Corr}(X,Y))}

Onde w1w2≤0,25w_1 w_2 \le 0{,}25, logo

12(1−Corr(X,Y))≤0,25\frac{1}{2(1 - \mathrm{Corr}(X,Y))} \le 0{,}25

4≤2(1−Corr(X,Y))4 \le 2(1 - \mathrm{Corr}(X,Y))

2−2Corr(X,Y)≥42 - 2 \mathrm{Corr}(X,Y) \ge 4

−2Corr(X,Y)≥2-2 \mathrm{Corr}(X,Y) \ge 2

Corr(X,Y)≤−1\mathrm{Corr}(X,Y) \le -1

Ou seja, dadas variâncias iguais, elas não são suficientes para que haja uma proporção de pesos que zere a variância da carteira.

Veja que

w1w2=12(1−Corr(X,Y))w_1 w_2 = \frac{1}{2\,(1 - \mathrm{Corr}(X,Y))}

−Corr(X,Y)=12w1w2−1-\mathrm{Corr}(X,Y) = \frac{1}{2\,w_1 w_2} - 1

Corr(X,Y)=1−12w1w2\mathrm{Corr}(X,Y) = 1 - \frac{1}{2\,w_1 w_2}

w1w2w_1 w_2 não pode ser menor que 0,250{,}25, pois isso implicaria uma |Corr(X,Y)|>1\lvert \mathrm{Corr}(X,Y) \rvert > 1

Ou seja, dada uma Corr(X,Y)=−1\mathrm{Corr}(X,Y) = -1, há somente uma combinação de pesos (sob a restrição de w1+w2=1w_1 + w_2 = 1), dadas variâncias iguais, que torna a variância da carteira igual a 00. Essa combinação é a em que w1=w2=0,5.w_1 = w_2 = 0{,}5.

Suponha Var(X)=kVar(Y)\mathrm{Var}(X) = k\,\mathrm{Var}(Y)

Como

Var(C)=w12Var(X)+w22Var(X)k+2w1w2Corr(X,Y)σXσY\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + \frac{w_2^2 \mathrm{Var}(X)}{k} + 2 w_1 w_2 \mathrm{Corr}(X,Y)\,\sigma_X \sigma_Y

Var(C)=w12Var(X)+w22Var(X)k+2w1w2Corr(X,Y)Var(X)k−1/2\mathrm{Var}(C) = w_1^2 \mathrm{Var}(X) + \frac{w_2^2 \mathrm{Var}(X)}{k} + 2 w_1 w_2 \mathrm{Corr}(X,Y)\,\mathrm{Var}(X)\,k^{-1/2}

Var(C)=Var(X)(w12+w22k+2w1w2Corr(X,Y)k−1/2)\mathrm{Var}(C) = \mathrm{Var}(X)\left( w_1^2 + \frac{w_2^2}{k} + 2 w_1 w_2 \mathrm{Corr}(X,Y)\,k^{-1/2} \right)

Impondo Var(C)=0\mathrm{Var}(C) = 0 e Corr(X,Y)=−1\mathrm{Corr}(X,Y) = -1,

w12+w22k−2w1w2k−1/2=0w_1^2 + \frac{w_2^2}{k} - 2 w_1 w_2 k^{-1/2} = 0

(w1−w2k−1/2)2=0(w_1 - w_2 k^{-1/2})^2 = 0

w1=w2k−1/2w_1 = w_2 k^{-1/2}

Sob a restrição w1+w2=1w_1 + w_2 = 1,

w2=k1/21+k1/2,w1=11+k1/2w_2 = \frac{k^{1/2}}{1 + k^{1/2}}, \quad w_1 = \frac{1}{1 + k^{1/2}}

Como k=Var(X)/Var(Y)k = \mathrm{Var}(X)/\mathrm{Var}(Y),

w2=σXσX+σY,w1=σYσX+σYw_2 = \frac{\sigma_X}{\sigma_X + \sigma_Y}, \quad w_1 = \frac{\sigma_Y}{\sigma_X + \sigma_Y}

Ou, equivalentemente,

w2=Var(X)Var(X)+Var(Y),w1=Var(Y)Var(X)+Var(Y)w_2 = \frac{\sqrt{\mathrm{Var}(X)}}{\sqrt{\mathrm{Var}(X)} + \sqrt{\mathrm{Var}(Y)}}, \quad w_1 = \frac{\sqrt{\mathrm{Var}(Y)}}{\sqrt{\mathrm{Var}(X)} + \sqrt{\mathrm{Var}(Y)}}

Para k=2k = 2,

w1=0,4142135624,w2=0,5857864376w_1 = 0{,}4142135624, \quad w_2 = 0{,}5857864376

Var(C)≈1,0×10−10\mathrm{Var}(C) \approx 1{,}0 \times 10^{-10}

Esse valor é apenas um resíduo numérico decorrente de aproximações.

Em síntese, sob a restrição w1+w2=1w_1 + w_2 = 1, é possível zerar a variância de uma carteira composta por dois ativos quando Corr(X,Y)=−1\mathrm{Corr}(X,Y) = -1.

Sendo ρXY=−1\rho_{XY} = -1, as expressões gerais dos cálculos dos pesos são:

w2=σXσY+σXw_2 = \frac{\sigma_X}{\sigma_Y + \sigma_X}

w1=σYσY+σXw_1 = \frac{\sigma_Y}{\sigma_Y + \sigma_X}

ou

w2=Var⁡(X)Var⁡(Y)+Var⁡(X)w_2 = \frac{\sqrt{\operatorname{Var}(X)}}{\sqrt{\operatorname{Var}(Y)} + \sqrt{\operatorname{Var}(X)}}

w1=Var⁡(Y)Var⁡(Y)+Var⁡(X)w_1 = \frac{\sqrt{\operatorname{Var}(Y)}}{\sqrt{\operatorname{Var}(Y)} + \sqrt{\operatorname{Var}(X)}}

Quando Var(X)=Var(Y)\mathrm{Var}(X) = \mathrm{Var}(Y), a única combinação que zera a variância é w1=w2=0,5.w_1 = w_2 = 0{,}5.
